# 项目简介

## 核心功能
- 系统集成百度千帆智能AI大模型
- 在线咨询功能
- 管理员、社长、用户三个角色
- 前后端代码及SQL文件
- 万字文档提供详尽说明
- 参考导入调试视频、讲解视频及开发工具安装包

## 适用场景
- 学校社团管理
- 社团活动组织与协调
- 会员信息管理
- 活动信息发布与通知

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
